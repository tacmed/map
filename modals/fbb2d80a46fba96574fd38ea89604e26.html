Заводские шины для иммобилизации.

Промышленность выпускает разные типы шин:

  • для конечностей
  • для шеи
  • для всего тела используется санитарный щит или вакуумный матрас (при множественных переломах)

Шины для конечностей изготавливают из картона (или листового пластика) и из металлической проволоки (по типу лестницы). Заводские шины имеют возможность по регулировке длины (люди-то разного размера). Шину всегда моделируют по здоровой конечности! Потом переносят на повреждённую.
